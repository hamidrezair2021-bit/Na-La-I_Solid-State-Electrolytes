The structures and codes relative to the manuscript titled 'Na-La-I Compounds as Solid-State Electrolytes for Sodium-Ion Batteries: A Combined DFT and MLIP Investigation' have been stored in this repository.

The structure folder lists the optimized unit cells of Na-La-I compounds.

convex_hull.py was used to calculate the energies above hull for each structure.

convex_hull_phase_plot.py was used for plotting phase plots from 'VASPRUN_SOURCE_hull_report.txt' which generated by convex_hull.py.

stability_window.py was used for plotting and calculating electrochemical stability windows of the solid-state electrolytes.

MD-analysis.py was used for calculating ion diffusions and ion conductivities
